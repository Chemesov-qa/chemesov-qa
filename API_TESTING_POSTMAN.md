## Что я использовал в тестах

На вкладке `Tests` каждого запроса я написал код на JavaScript, который:

- **Сохраняет ID брони** после создания — чтобы использовать в следующих запросах
- **Проверяет коды ответов** — 200, 201, 403, 404
- **Сравнивает данные** — что отправили и что вернул сервер
- **Проверяет структуру JSON** — все ли поля на месте
- **Анализирует текст ошибок** — содержит ли ответ слово "Forbidden"

Пример кода из тестов:

```javascript
// Сохраняем ID брони для следующих шагов
const response = pm.response.json();
pm.environment.set("bookingId", response.bookingid);

// Проверяем что код ответа 200
pm.test("Статус код 200", function () {
    pm.response.to.have.status(200);
});

// Проверяем что данные совпадают
pm.test("Данные в базе верны", function () {
    pm.expect(response.firstname).to.eql("Влад");
});
