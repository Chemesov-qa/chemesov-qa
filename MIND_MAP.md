# 🧠 Интеллект-карта проекта (Mind Map)

Визуализация декомпозиции системы «Кордон» для определения стратегии тестирования. Данная карта помогла структурировать проверки и обеспечить 100% покрытие критических модулей.

#Использование Mind Map позволило мне визуализировать структуру комплекса «Кордон» и выделить критические узлы для глубокой проверки. 
Это сократило время на проектирование тестов и гарантировало 100% покрытие функциональных требований.

### 🔍 Зачем это нужно в тестировании:
1. **Декомпозиция:** Сложная система разбита на простые проверяемые модули.
2. **Визуализация рисков:** Красным цветом на карте выделены критические узлы (Связь, БД), для которых написаны детальные [тест-кейсы](./TEST_CASES.md).
3. **Анализ покрытия:** Карта гарантирует, что при регрессионном тестировании не будет упущен ни один компонент (например, проверка прав доступа или работа плагинов).

### 🛠 Инструмент:
Карта создана в **XMind**. Это позволило быстро менять приоритеты проверок в зависимости от изменений в ТЗ.
